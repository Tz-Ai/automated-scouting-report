# Automated Scouting Report Generator

An automated esports analytics dashboard that transforms historical match data into actionable scouting insights.

## Features
- Map-based win rate analysis
- Player performance breakdown
- Draft pick and ban tendencies
- Win conversion and loss collapse metrics
- Actionable "How to Win" recommendations

## Project Structure
- `app.py`: Streamlit dashboard
- `scripts/`: Data processing and helper logic

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- GRID Esports Data

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
